RC PPM Decoder
==============

Here is an sample application to decode RC's PPM signal
Originally created for [Virt2real](http://virt2real.ru)

How to build
------------

1. Copy `settings-template.mk` as `settings.mk`
2. Edit settings.mk to adjust variables
3. Run `make`
